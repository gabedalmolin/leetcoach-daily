# LeetCoach Daily

LeetCoach Daily is a learning-first companion for the LeetCode Daily problem.

Instead of dumping a full answer, it helps you reason about the problem:
- identify the likely pattern
- understand the target time and space complexity
- receive progressive hints
- walk through the examples step by step
- review your own solution and explain likely mistakes

## Why it exists

Most tools optimise for speed or convenience.
LeetCoach Daily optimises for understanding.

The goal is simple: help you solve the Daily problem yourself, with just enough guidance at the right time.

## Core features

- Pattern detection
- Complexity target
- Progressive hint ladder
- Example walkthrough
- Solution review
- Personal learning history

## Product principles

- Learning first
- No forced spoilers
- Explainable suggestions
- Deterministic core
- Clear feedback over flashy output
